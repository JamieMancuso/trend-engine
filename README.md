\# Trend Engine



Personal research digest that surfaces emerging technological paradigm shifts

across 8 research domains (AI, Bio, Health, Space, Quantum, Robotics, Energy,

Climate) for a 2-year retail investment horizon.



Pulls papers from arXiv, scores them via Claude API against a maturation /

profit / accessibility / specificity rubric, and renders a daily digest as a

Streamlit app.
